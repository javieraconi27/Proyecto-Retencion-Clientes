# Dashboard de Gestión de Retención de Clientes

**Autor:** [Javiera Cerda]  
**Curso:** Taller de Inteligencia de Negocios - UDD  
**Link al Dashboard:** [https://javieracerda.shinyapps.io/Proyect-TIN/]

## Descripción
Este proyecto implementa una solución de Business Intelligence para predecir y gestionar la fuga de clientes. Incluye modelos de regresión logística y segmentación (clustering).

## Archivos
- `app.R`: Código fuente.
- `synthetic_customers_churn.csv`: Base de datos.
